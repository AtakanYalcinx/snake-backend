# Project Snake-Backend

Backend  für  das Snake  Game. Es  gibt verschiedene  Modi die Eingestellt werden können.



## Technologien

- Java (Spring Boot)
- Gradle
- Docker
- Render (Deployment)


## Voraussetzungen


- Java 21
- Gradle 8.7
- Docker


## Projekt  Starten

- docker-compose up
-  gradle build
- SnakeBackendApplication  starten
